## Quick Links

### Security Features
- ✅ Governance token locking (H-03/H-04 fix)
- ✅ Distribution token locking (H-02 fix)
- ✅ Operator voting restrictions (H-05 fix)
- ✅ Pro-rata distributions (M-01 fix)
- ✅ Reentrancy protection
- ✅ O(1) scalability (10,000+ holders)

### Proposal Types
- Resolution proposals (text-only governance)
- Treasury proposals (asset transfers)
- Mint proposals (token creation)
- Parameter proposals (configuration changes)
- Distribution proposals (proportional distributions)
